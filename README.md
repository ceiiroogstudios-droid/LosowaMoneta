# LosowaMoneta (RandomCoin)
Description: Tired of regular coins that do nothing? With this plugin, when a player uses a coin, there's a chance of:
- a random item
- a random effect
- switching the player to a random role
- removing equipment
- pink candy

The plugin is in Polish, but you can change the translation of hints in the config.
